# BookAndOrder
A E-commerce web application built with ASP.NET Core 9.

## Stack
* **ASP.NET Core 9 (MVC)**
* **ASP.NET Core Identity** (Roles & Authentication)
* **Entity Framework Core**
* **SQL Server**
* **Bootstrap & Bootstrap Icons**
* **Unsplash** (Dynamiska produktbilder)

## Features
* **Product Catalog:** Browse products with search and category filters.
* **Shopping Cart:** Add or remove items. The total price is calculated automatically.
* **User Accounts:** Customers can see their profile and order history.
* **Admin Mode:** Admins can Create, Edit, and Delete products and categories.
* **Responsive Design:** Works on both desktop and mobile.
* **Database Seeding:** The database is automatically filled with 14 products and an Admin account on first setup.

## Installation

### 1. Clone the repository
git clone [https://github.com/AlbinWr/BookAndOrder.git](https://github.com/AlbinWr/BookAndOrder.git)
cd BookAndOrder

### 2. Update the Database
dotnet ef database update

### 3. Run the project
dotnet run
